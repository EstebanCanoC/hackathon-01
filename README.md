Crear una landing page interactiva, que muestre los principales productos y servicios de una tienda deportiva. La plataforma online debe destacarse por:

Tener un diseño atractivo.
Facilitar la exploración del contenido mediante una interfaz intuitiva. 
Adaptarse visualmente a diferentes dispositivos. 

Usarán las siguientes herramientas:

HTML (HyperText Markup Language): Para crear y estructurar el contenido de páginas web.
Hojas de Estilo (CSS o Bootstrap): Para controlar el aspecto y la presentación de un sitio web.
Animaciones con JavaScript: Aplica animaciones, inserciones de los productos al carrito y transiciones de página para enfatizar ideas y hacerlas incluso más memorables
Visual Studio Code: Para desarrollo del proyecto que incluya todos los lenguajes
